📞 Call Center Performance Dashboard
🔍 Overview
Analyzed call center operations to identify key factors impacting customer satisfaction (CSAT) and service performance.

🎯 Objective
Understand reasons behind declining CSAT
Identify key drivers such as sentiment, issue type, and call duration
Provide actionable insights to improve service quality
🛠️ Tools Used
Power BI – Dashboard and visualization
SQL – Data analysis and KPI calculation
Power Query – Data cleaning and transformation
