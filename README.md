# 📞 Call Center Performance Dashboard

## 🔍 Overview
Analyzed call center operations to identify key factors impacting customer satisfaction (CSAT) and service performance.

---

## 🎯 Objective
- Understand reasons behind declining CSAT  
- Identify key drivers such as sentiment, issue type, and call duration  
- Provide actionable insights to improve service quality  

---

## 🛠️ Tools Used
- Power BI – Dashboard and visualization  
- SQL – Data analysis and KPI calculation  
- Power Query – Data cleaning and transformation

## 📊 Key Metrics
- Total Calls  
- Average Call Duration  
- SLA %  
- CSAT Score  
- Sentiment Distribution  

---

## 🔎 Key Insights
- High negative sentiment significantly impacts CSAT  
- Billing-related queries have the lowest satisfaction scores  
- Longer call durations are associated with lower CSAT  

---

## 💡 Recommendations
- Improve billing issue resolution process  
- Optimize call handling time through agent training  
- Monitor channel performance to identify weak areas  

---

## 📈 Expected Impact
- Improved CSAT scores  
- Reduced customer dissatisfaction  
- Better operational efficiency
